# PS网页版

这是一套功能极其强大的在线图像处理系统，它打破了传统软件的桎梏，让用户无需安装任何客户端，仅通过浏览器即可享受专业级的图像编辑体验，是搭建在线设计工具站的绝佳选择。

##  功能与优势：

- 专业级编辑环境：界面布局与桌面版Photoshop高度一致，包含图层管理、工具栏、属性面板等核心模块。无论是专业的UI设计师还是普通用户，都能零门槛上手，无需重新学习操作逻辑。

- 全格式文件兼容：完美支持打开和编辑`.PSD、.AI、.XD、.Sketch、.PDF、.RAW`等多种专业设计格式，同时也支持JPG、PNG、SVG等通用图片格式。它不仅能修图，还能作为格式转换器使用，极大地扩展了应用场景。

- 强大的图层系统：内置完整的图层操作功能，支持图层样式、蒙版、混合模式、矢量蒙版等高级特性。用户可以像在本地软件中一样，对图层进行自由移动、缩放、编组和调整，确保复杂的合成设计也能精准完成。

- 丰富的设计工具：提供包括移动工具、选框工具、画笔、仿制图章、文字工具、裁剪工具等在内的全套设计工具。无论是照片精修、海报设计、还是简单的图片裁剪，都能游刃有余。

- 零环境依赖部署：系统架构极其轻量，经测试，只需将源码放置于服务器任意目录，通过Nginx进行反向代理即可直接运行。无需配置复杂的PHP、MySQL等后端环境，大大降低了站长的运维成本和部署难度。

## 部署与使用说明

- 部署方式：无需额外环境（如PHP/MySQL），直接将源码文件放到服务器任意位置。

- 运行配置：推荐使用Nginx进行反向代理，配置成功后即可正常访问使用。
功能体验：功能与客户端版PS几乎一模一样，该有的功能都有，支持拖拽上传文件。

- 支持格式：`.PSD, .AI, .XD, .FIG, .sketch, .PDF, RAW, JPG, PNG, GIF, TIFF, SVG, DDS, ANY`。
